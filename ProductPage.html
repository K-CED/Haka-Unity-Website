App Specifications

OPERATING SYSTEM:       Android 4.1 and Up

WHAT IS AN OPERATING SYSTEM?

An Operating System (OS) is responsible for running applications, specifically for mobile devices. An example of an operating system is Android which is an open source software, meaning that the user can customize the architecture and the flow of the system. When we say customizing, this can be done by installing a custom firmware for your phone. An Operating System is a very important component inside a device. Without this, your smartphone will not function the way it should be.

RAM:       500MB and Up

WHAT IS A RAM?

A Random Access Memory (RAM) is where an application or software resides while it is running, along with the data it is using. Both the OS and application software use RAM. Devices with more RAM can run more complex software and/or more applications at one time. Moving data into and out of RAM takes time. If data needs to be moved out of RAM to make room for other data, this can bring everything to a halt for a moment.

SoC / CHIPSET:       TI OMAP 4460 or Greater.

WHAT IS A CHIPSET?

A chipset is commonly known as a "Processor". This is responsible for everything in your smartphone such as booting up, running apps, executing commands, utilization of biometrics and many more. There are many types of processors for mobile phones. Examples are: Kirin, Mediatek, Exynos, Snapdragon, and many more. High-end chipsets maintain the device from overheating because it surpasses the high-processing power of applications.

ROM / STORAGE:       Atleast 1GB or More.

WHAT IS A ROM?

A ROM stores files from your device. Think of it as a container that stores data virtually. Each device has different storage sizes. It can be 8GB, 16GB, 64GB, 512GB, and even 1TB of storage. Shutting down the device does not format or delete your saved files. Both ROMS can be external or internal. Internal storage built within the phone itself. You can not expand the size of this but you can get an external storage such as an "SD Card" so you can expand your phone's storage and eventually insert additional files as you wish.


Pro Tip: If you want to know more about the specs of your device, you can download "CPU-Z" from the Google Play Store for free. This app states the full specs of your phone not just the storage, cpu, gpu, and ram.
